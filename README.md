# portal-site-archive

本仓库用于归档和发布多个独立 HTML 页面。  
这些页面可能来自不同时期、不同主题，彼此之间没有强关联。  
仓库不对应任何特定域名或网站，仅作为静态 HTML 页面的统一存档与展示。

## 项目简介

`portal-site-archive` 是一个以归档为目的的静态页面集合。  
每个 HTML 文件均为完整可独立打开的页面，不依赖后台服务或数据库。  
仓库本身不提供搜索、评论或用户系统，仅作为内容存档的“容器”。

## 目录说明

```
.
├── index.html          # 首页（可选），用于列出归档页面链接
├── pages/              # 存放所有归档 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源（CSS、图片、字体等）
│   ├── css/
│   ├── images/
│   └── ...
├── README.md           # 本文件
└── LICENSE             # 开源许可证（如有）
```

- `pages/` 目录下的每个 HTML 文件即为一条归档记录。
- `assets/` 目录存放页面引用的公共资源，便于维护与复用。
- `index.html` 为可选入口页面，可用于快速浏览所有归档页面。

## 页面归档说明

- 所有页面均为静态 HTML，原则上不包含外部依赖（除本仓库 assets 中的资源）。
- 页面内容可能包含文本、图片、链接等，但不保证内容的时效性或准确性。
- 归档页面可能来自不同时期，格式、风格、语言可能不统一。
- 部分页面可能包含过时或失效的链接，请以存档内容为准。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会通过 commit 记录。
- 欢迎提交 Issue 或 Pull Request，但请确保内容符合本仓库的归档定位。
- 对于明显无关或违反法律法规的内容，维护者有权拒绝合并或直接删除。
- 本仓库不提供自动化部署或持续集成配置，页面可直接通过 GitHub Pages 或其他静态托管服务发布。
